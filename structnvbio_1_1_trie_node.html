Detailed description

template< TrieType TYPE_T>
struct nvbio::TrieNode< TYPE_T >

A suffix trie node

Template Parameters
TYPE_Ta suffix trie can be stored either with an uncompressed layout, where each inner node reseves storage for |alphabet| children and some of them are marked as invalid, or with a compressed one, where only storage for the active children is actually reserved; the latter type requires just a little more logic during traversal, as a popcount is necessary to select the child corresponding to the i-th character.
